# Metro Capacity Monitoring System 🚆

## Description
This project simulates a metro passenger capacity monitoring system using Arduino Uno. It tracks the number of passengers entering and exiting and displays real-time occupancy on an LCD.

## Components Used
- Arduino UNO
- 16x2 LCD Display
- Push Buttons (Entry & Exit)
- LED Indicator
- Resistors & Breadboard
(Refer to bom.csv)

## Features
- Real-time passenger count
- Entry and exit tracking using buttons
- LCD display output
- LED indication when capacity is full
- Overflow and underflow protection included
